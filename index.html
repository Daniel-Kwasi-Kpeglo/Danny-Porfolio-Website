What I do

Data Analysis with Python

Exploratory Data Analysis (EDA) techniques are used on primary datasets as well as data obtained from Kaggle to show the relationship between the datasets. The data obtained from Kaggle is further cleaned and python libraries such as pandas, NumPy, matplotlib, plotly, seaborn, are used to produce graphical images of the datasets for further visualization purposes. Examples of projects include; 911 calls assessment, Eccomerce assessment, Data Capston Financial Project, e.t.c.

Data Cleaning with SQL

Data for projects under this section is obtained from Kaggle and "Our World in Data" platform. SQL clauses such as Select, Where, Order by, Group by, and other query statements are used to clean the datasets. Populated columns are broken into individual columns, duplicates are removed, and new columns are created just to mention a few actions. Examples of projects under this category include; Nashville Housing Data Cleaning Project, Covid Global Infection rate Project.

Data Dashboards

Dashboards using Tableau and Microsoft Power BI are created after the Data have been thoroughly cleaned using SQL. The dashboards tracks, analyzes, display KPIs, the relationships between the datasets, and show critical data points detailing the impact of the data under consideration. rate using Tableau.
